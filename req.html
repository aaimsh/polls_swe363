REQUIREMNTS

  • REQ01:The System shall allow visitors to signup by using their email address and no duplicate emails should be allowed if a visitor entered email address that already that exists in the database the System shall display a message states this email is registered .
  • REQ01: The System shall allow users to login using their email & password that match the values in the database .
  • REQ02: The System shall allow logged in member to create new polls by creating question and options for each question .
  • REQ03:The System shall allow visitors with valid URL to an existing poll to vote by choosing only one option for each question .
  • REQ04: The System shall provide real-time results for each question to the poll creator .
  • REQ05: The System shall provide a unique URL for each poll and display it to the creator to share it with the voters .
